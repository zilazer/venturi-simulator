# Venturi Simulator / 文丘里管水气混合仿真工具

一个单文件静态网页工具，用于小型文丘里水气混合、曝气、喷射器和入口曲线方案的工程初选。当前版本不依赖后端、不需要构建步骤，核心文件是 `index.html`。

在线访问：

- GitHub Pages: https://zilazer.github.io/venturi-simulator/
- GitHub Repo: https://github.com/zilazer/venturi-simulator

> 重要说明：本工具是工程初选和趋势判断模型，不是 CFD，不替代实验标定、制造级校核、压力容器/管路安全设计或正式产品认证。

## Quick Start

1. 在线使用：打开 https://zilazer.github.io/venturi-simulator/
2. 本地使用：下载仓库后，直接用浏览器打开 `index.html`。
3. 更新网页：修改 `index.html` 后提交并推送到 `main`，GitHub Pages 会自动更新。

## 功能页面

### 仿真页

用于输入进水流量、入口压力、空气入口、背压和环境参数，自动推荐文丘里管几何参数，并输出吸气、背压、空化和混合强度状态。

主要输出：

- 推荐入口直径、喉径、出口直径、收缩半角、扩散半角、喉部长度、进气口直径。
- 喉部压力、速度、出口速度、总压损、气水比、估算含气率。
- 管内剖面动画、出口流态动画、异常状态参数调整箭头。

### 目标反算页

用于输入目标进水量和目标进气量，自动寻找一个可吸气、背压可克服的方案。适合从“我要达到某个流量目标”出发做初步选型。

主要能力：

- 支持目标进气量单位 `L/min` 与 `mL/min`。
- 支持期望进水口直径，并将入口直径偏差纳入搜索评分。
- 可一键把目标反算结果复制到手动校核页继续验算。

### 手动校核页

用于手动输入完整工况和文丘里几何参数，程序不自动改动几何，只判断当前设计状态。

适合验证：

- 指定入口直径、喉径、出口直径是否能吸气。
- 指定收缩半角、扩散半角、进气口位置、进气口直径对压损和背压能力的影响。
- 高背压、逆流、空化和混合强度风险。

### 深度计算页

基于手动几何做 1D 多节点工程校核，把文丘里管分为入口、收缩段、喉部、混合段、扩散段和出口多个节点。

主要输出：

- 沿程压力、速度、含气率、混合密度、空化裕度。
- 最低压力、最低空化裕度、最大含气率、出口可用压力。
- 沿程曲线和节点表格。
- 深度计算 Markdown 报告导出。

### 工程口诀页

整理文丘里和微纳米气泡模块的工程经验，用于实验调参时快速判断方向。

覆盖主题：

- 基础流动规律。
- 喉径、喉长和扩散段经验。
- 进气侧阻力、止回阀、气路弯头。
- 微纳米气泡、空化、剪切和系统级经验。

### 入口曲线页

用于比较锥形入口和曲线漏斗入口。曲线入口使用光顺收缩曲线生成，重点展示更平滑的壁面斜率对局部损失、分离风险和流动稳定性的影响。

主要能力：

- 输入入口直径、喉径、入口段长度、曲线指数等参数。
- 输出锥形与曲线入口的估算损失差异。
- 导出曲线坐标和 STP 文件。

## 导出能力

- STP：从仿真页、目标反算页、手动校核页和入口曲线页导出简化几何文件。
- MD：从仿真、目标反算、手动校核和深度计算结果导出计算步骤报告。
- 曲线数据：入口曲线页可生成曲线坐标，用于后续建模参考。

## 计算方法总览

更完整的公式说明见 [docs/calculation-methods.md](docs/calculation-methods.md)。

### 共用物理假设

- 水侧按不可压缩流体处理。
- 空气侧使用简化可压缩孔口流量模型，并给出临界流提示。
- 两相流结果用于趋势判断，不做 CFD 级相间耦合。
- 温度影响水密度、水黏度、水蒸气压和空气密度。
- 所有页面内部优先使用 SI 单位，显示时再转换为 `kPa`、`MPa` 或 `kgf/cm^2` 等工程单位。

### 参数确认方式

- 几何参数来自自动搜索、目标反算结果或用户手动输入。
- 收缩半角和扩散半角会参与局部损失、压恢复和分离风险判断。
- 进气口轴向位置会影响负压利用效率和吸气能力。
- 第三方模型允许调整损失系数、压恢复效率、气水比阈值、空化裕度等模型参数。

### 计算与验算方式

- 连续方程计算各截面速度。
- 伯努利方程和局部损失估算喉部压力与总压损。
- 空气孔口流量估算吸气量或反算进气口直径。
- 喉部压力低于空气入口压力时判定有吸气条件。
- 混合段或喉部压力高于空气入口压力且背压过高时提示进气口逆流风险。
- 喉部绝压接近水蒸气压时提示空化风险。
- 出口可用压力低于背压时提示背压不可克服。
