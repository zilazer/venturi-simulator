# Changelog

本文件记录文丘里仿真工具的主要更新。格式采用“最新在前”。

## Unreleased

- 新增项目 README、计算说明、路线图、验收清单和 GitHub 协作模板。

## 2026-05-29

### Added

- 入口曲线页增加 STP 导出能力。

## 2026-05-28

### Added

- 增加入口曲线对比页，用于比较锥形入口和曲线漏斗入口。
- 增加中文/EN 语言切换。
- 增加固定状态栏，使吸气状态、出口背压、空化判断和混合强度在计算页滚动时保持可见。
- 增加目标反算结果复制到手动校核页的工作流。
- 增加仿真页、目标反算页、手动校核页、深度计算页、工程口诀页等核心功能。
- 增加 STP 和 Markdown 计算报告导出能力。

### Changed

- 校准公式面板，使每页展示的公式与实际计算路径一致。
- 优化目标反算页对低流量、小入口直径场景的搜索。
- 调整页面视觉设计和导航结构。

### Fixed

- 修复目标反算复制到手动校核页时 `0` 值可能被当作空值覆盖的问题。
