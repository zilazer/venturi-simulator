# 计算方法与参数验算

本文档说明当前网页中的主要工程计算方法。所有模型都用于工程初选和趋势判断，不是 CFD，也不替代实验标定。

## 单位与物性

内部计算优先使用 SI 单位：

- 流量：`m^3/s`
- 压力：`Pa`，页面显示可切换为 `kPa`、`MPa`、`kgf/cm^2`
- 直径与长度：内部 `m`，页面显示 `mm`
- 温度：`deg C`

水温用于估算：

- 水密度 `rho_w`
- 水动力黏度 `mu_w`
- 水蒸气压 `p_vapor`

空气温度和大气压用于估算：

- 空气密度 `rho_air`
- 空气入口绝压 `p_air,in`

## 共用公式

### 连续方程

```text
A = pi * D^2 / 4
v = Q / A
```

含义：

- `A` 为截面积。
- `D` 为截面直径。
- `Q` 为体积流量。
- `v` 为平均流速。

用途：

- 根据进水量和入口、喉部、出口直径计算各截面水速。
- 直径越小，同流量下速度越高。

### 伯努利压降与局部损失

```text
Delta p_loss = K * 0.5 * rho * v^2
p_t ~= p_in + 0.5 * rho * (v_in^2 - v_t^2) - Delta p_loss
```

含义：

- `K` 为局部损失系数。
- `rho` 为流体密度。
- `p_t` 为喉部压力。
- `v_in` 为入口速度。
- `v_t` 为喉部速度。

用途：

- 估算喉部负压是否足够吸气。
- 收缩半角、表面粗糙度、喉部长度和进气口位置会影响损失项。

### 雷诺数

```text
Re = rho * v * D / mu
```

含义：

- `Re` 表示惯性力和黏性力的相对强弱。
- `mu` 为动力黏度。

用途：

- 判断流动状态。
- 为摩擦因子估算提供依据。

### 摩擦因子近似

```text
f ~= 64 / Re                     laminar
f ~= 0.25 / [log10(eps/(3.7D) + 5.74/Re^0.9)]^2    turbulent
```

含义：

- `f` 为 Darcy 摩擦因子。
- `eps` 为等效粗糙度。

用途：

- 估算喉部长度、入口段、混合段和扩散段带来的沿程损失。

### 压恢复

```text
p_out,available ~= p_t + eta_r * 0.5 * rho * (v_t^2 - v_out^2) - Delta p_diffuser
```

含义：

- `eta_r` 为扩散段压恢复效率。
- `Delta p_diffuser` 为扩散段附加损失。

用途：

- 判断出口可用压力是否能克服背压。
- 扩散半角过大时，分离风险提高，压恢复效率降低。

### 空气孔口流量

简化不可阻塞近似：

```text
Q_air ~= C_d * A_air * sqrt(2 * Delta p / rho_air)
```

临界流提示：

```text
p_down / p_up <= (2/(gamma+1))^(gamma/(gamma-1))
```

含义：

- `C_d` 为孔口流量系数。
- `A_air` 为进气口面积。
- `Delta p` 为空气入口与喉部或混合区之间的压差。
- `gamma` 为空气绝热指数。

用途：

- 在给定进气孔径时估算进气量。
- 在给定目标进气量时反算进气口等效直径。
- 当压比达到临界条件时，只给出“可能临界流”提示，不处理复杂激波。

### 气水比与含气率

```text
R_g = Q_air / Q_water
alpha = Q_air / (Q_air + Q_water)
```

含义：

- `R_g` 为气水比。
- `alpha` 为估算体积含气率。

用途：

- 判断混合强度。
- 气水比过高时提示两相模型误差增大。

### 空化裕度

```text
margin_cav = p_abs - p_vapor
```

含义：

- `p_abs` 为局部绝压。
- `p_vapor` 为对应水温下的水蒸气压。

用途：

- 当局部绝压接近水蒸气压时提示空化风险。
- 工业喷射器模型采用更保守的空化裕度阈值。

### 背压判定

```text
p_out,available >= p_back
```

用途：

- 若出口可用压力低于背压，提示背压不可克服。
- 高背压会缩短出口射流动画长度，并提高逆流风险。

### 逆流判定

```text
p_mix_or_throat >= p_air,in
```

用途：

- 当喉部或混合段压力高于空气入口压力时，空气无法被吸入。
- 若背压进一步推高混合区压力，提示进气口倒灌或逆流风险。

## 页面计算路径

### 仿真页

仿真页从水侧流量、入口压力、空气输入、背压和环境参数出发，搜索推荐几何。

计算流程：

1. 根据进水流量和候选喉径计算喉部速度。
2. 使用伯努利方程和损失系数估算喉部压力。
3. 判断喉部压力是否低于空气入口压力。
4. 根据空气输入模式估算进气量或反算进气口直径。
5. 估算压恢复和出口可用压力。
6. 输出吸气、背压、空化、混合强度状态。

参数确认：

- 入口直径、出口直径、喉径、喉长和角度由搜索和模型配置推荐。
- 收缩半角和扩散半角会随流速、压差、气水比、粗糙度和模型配置变化。
- 用户锁定或手动输入的参数优先参与计算。

验算方式：

- 检查喉部压力是否低于空气入口压力。
- 检查出口可用压力是否高于背压。
- 检查空化裕度是否高于模型阈值。
- 检查气水比是否进入模型高误差区。

### 目标反算页

目标反算页从目标进水量和目标进气量出发，寻找可吸气方案。

计算流程：

1. 枚举入口压力和喉径候选。
2. 计算每个候选的喉部压力、速度、空化裕度和出口可用压力。
3. 反算达到目标进气量所需的进气口等效直径。
4. 将期望入口直径偏差、压损、气水比、空化裕度和孔径合理性纳入评分。
5. 选择吸气成立、无逆流、背压可克服且评分最低的方案。

参数确认：

- 目标水量和目标气量由用户输入。
- 期望进水口直径作为评分项，不是硬性几何覆盖。
- 搜索范围由当前应用模型和第三方模型参数决定。

验算方式：

- 找到方案后应复制到手动校核页，确认同一几何下仍为可吸气。
- 对低流量场景，重点检查孔径是否过小、加工精度是否可实现。

### 手动校核页

手动校核页只判断用户输入的几何，不自动修改参数。

计算流程：

1. 使用用户输入的入口直径、喉径、出口直径计算各截面速度。
2. 使用用户输入的收缩半角、扩散半角、喉长和进气位置计算损失与压恢复。
3. 根据空气输入和进气孔径估算吸气状态。
4. 输出背压、逆流、空化和混合强度判断。

参数确认：

- 所有几何参数均以用户输入为准。
- 模型切换只改变损失系数、压恢复效率和风险阈值，不改几何。

验算方式：

- 手动改变喉径、出口直径、角度和进气口位置，观察状态卡与调整箭头。
- 高背压或逆流时，重点检查出口压力恢复和进气口附近压力。

### 深度计算页

深度计算页使用 1D 多节点模型，不做 CFD。

计算流程：

1. 根据手动几何生成入口、收缩段、喉部、混合段、扩散段和出口节点。
2. 每个节点计算直径、截面积和水速。
3. 逐段计算压力损失和压恢复。
4. 可选考虑空气体积随压力膨胀：

```text
Q_air(x) = Q_air,in * p_air,in / p(x)
```

5. 可选考虑两相混合密度：

```text
rho_mix = alpha * rho_air + (1 - alpha) * rho_water
```

6. 每个节点计算含气率和空化裕度。

验算方式：

- 增加节点数后，出口压力和最低压力应保持同量级。
- 高背压场景应在风险列表中标记背压或逆流风险。
- 低压场景应标记最低空化裕度节点。

### 工程口诀页

工程口诀页不参与数值求解，是调参经验整理。

用途：

- 帮助判断异常状态的调整方向。
- 连接几何参数与实验现象，例如“喉径决定吸得进，喉长决定打得碎”。

### 入口曲线页

入口曲线页比较锥形入口和曲线漏斗入口。

曲线形式：

```text
r(x) = r_t + (r_in - r_t) * [1 - S_p(x/L)]
```

其中 `S_p` 是双端光顺的 S 型收缩函数。

计算意义：

- 锥形入口壁面斜率突变更明显，局部损失和分离风险通常更高。
- 曲线漏斗在入口和喉部连接处更光顺，通常更利于稳定加速。

生成方式：

- 根据入口直径、喉径、入口段长度和曲线指数生成半径坐标。
- 将二维曲线旋转生成简化轴对称几何。
- STP 导出为工程预览几何，后续制造仍建议在 CAD 中重建并校核尺寸。

## 模型限制

- 当前模型没有求解真实三维湍流。
- 没有处理复杂激波、非平衡空化、真实气泡破碎谱或氧转移动力学。
- STP 导出是简化几何，不是完整制造图纸。
- 低流量、微小孔径和高含气率场景对加工误差、气路阻力和止回阀非常敏感，需要实验标定。
