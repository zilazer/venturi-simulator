# 发布前验收清单

用于每次推送 GitHub Pages 前后的人工检查。

## 页面结构

- [ ] GitHub Pages 页面可以打开，无空白屏。
- [ ] 顶部导航可以切换六个页面：仿真页、目标反算页、手动校核页、深度计算页、工程口诀页、入口曲线页。
- [ ] 页面在桌面宽度下没有明显重叠。
- [ ] 页面在手机宽度下输入区、输出区和 Canvas 不互相遮挡。

## 通用功能

- [ ] 压力单位可在 `kPa`、`MPa`、`kgf/cm^2` 之间切换。
- [ ] 中文/EN 切换后主要按钮、标题和说明可读。
- [ ] 计算页固定状态栏随页面滚动保持可见。
- [ ] 状态卡 hover 时能显示当前状态解释。
- [ ] 异常状态下，相关几何参数框显示调整方向箭头。

## 仿真页

- [ ] 修改进水流量、入口压力、背压、水温后，输出实时更新。
- [ ] 进气模式切换后，目标进气量和空气孔径计算正常。
- [ ] STP 导出按钮可下载文件。
- [ ] Markdown 计算步骤导出按钮可下载文件。
- [ ] “复制到手动校核页”能带入当前方案。

## 目标反算页

- [ ] 输入目标进水量和进气量后能找到可吸气方案。
- [ ] `L/min` 与 `mL/min` 切换正常。
- [ ] 期望进水口直径会影响搜索结果评分。
- [ ] 示例工况 `入口直径 3.5 mm`、`进水 0.6 L/min` 能找到可吸气且背压可克服的方案。
- [ ] “复制到手动校核页”后，手动页保持同一几何并得到一致状态。

## 手动校核页

- [ ] 手动输入入口直径、喉径、出口直径、角度、喉长、进气口直径和进气位置后能输出状态。
- [ ] 改变收缩半角或扩散半角会影响压损、压恢复或风险提示。
- [ ] 高背压场景能显示背压过高或逆流风险。
- [ ] 喉部低压场景能显示空化风险。

## 深度计算页

- [ ] 可从手动校核页填入几何参数。
- [ ] 节点数从 12 改到 48 后不出现 `NaN`。
- [ ] 压力、速度、含气率曲线非空。
- [ ] 深度计算 Markdown 文件可下载。

## 工程口诀页

- [ ] 口诀内容正常显示。
- [ ] 中英文切换后页面仍可读。

## 入口曲线页

- [ ] 可生成锥形入口和曲线入口对比。
- [ ] 曲线图非空。
- [ ] 曲线坐标可用于后续 CAD 建模参考。
- [ ] STP 导出按钮可下载文件。

## 仓库文档

- [ ] README 在 GitHub 仓库首页正常渲染。
- [ ] README 内部链接可点击。
- [ ] CHANGELOG 最新条目在最上方。
- [ ] `docs/calculation-methods.md` 与网页公式说明不冲突。
